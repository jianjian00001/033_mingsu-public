## 基于SpringBoot+vue的民宿预定管理系统(源代码+数据库+7500字文档)033

## 一、系统介绍

前台：

- 民宿预定、我的预定。       
- 民宿周边美食、民宿周边美景。      
- 民宿交流。
- 登录、注册、修改密码

后台管理：

- 民宿管理
- 民宿周边美食管理、民宿周边美景管理
- 订单管理
- 用户管理

## 二、所用技术

### 后端

* 核心框架：SpringBoot
* 持久层框架：MyBatisPlus
* 权限框架：SpringSecurity

### 前端

vue

## 三、环境介绍

基础环境 :IDEA/eclipse, maven3.x, JDK 1.8 , Mysql, Node.js

源码+数据库脚本
